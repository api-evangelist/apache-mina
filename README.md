# Apache MINA (apache-mina)
Apache MINA is a network application framework that helps develop high-performance and high-scalability network applications. It provides an abstract event-driven asynchronous API over various transports such as TCP/IP and UDP/IP via Java NIO.

## APIs

### Apache MINA
MINA provides a Java NIO-based API for building network applications with support for TCP, UDP, and serial protocols, along with filter chain API for protocol codecs and the SSHD sub-project for SSH client/server implementation.
